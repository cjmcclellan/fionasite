Ultra-thin Memory Selectors

{% endblock %} {% block cardbody %}
In this era of data-driven applications, memory technology has become increasingly important. In order to achieve a high density memory array, two-terminal selectors are necessary. However, commonly used threshold-type selectors suffer from many issues originated from their ion/atom motion-based mechanism. Non-threshold type selectors are based on reliable tunneling, however, the traps in oxide limit the endurance and current drive. 2D materials are the best candidates to replace oxide.
